How website management works now

RD Advance Boutique is a fully static website. Update content by editing JSON files in the repository and committing the changes.

  • Products: edit data/products.json to add, edit, or delete products.
  • Gallery: edit data/gallery.json to add, edit, or delete designs.
  • Categories: edit data/categories.json.
  • Settings: edit data/settings.json for branding, contact, SEO, and social links.
  • Notifications: edit data/notifications.json.

No login, sessions, server-side storage, Express backend, or JSON write operations are used. Customer product orders, gallery enquiries, custom stitching requests, course enquiries, and contact messages all open WhatsApp Business with prefilled messages.
