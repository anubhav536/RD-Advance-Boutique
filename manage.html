⚙️ Google Sheets Setup Guide

1
Create a Google Sheet

Go to sheets.google.com → New Sheet → name it RD Boutique Orders

2
Open Apps Script Editor

In the sheet menu: Extensions → Apps Script → Delete the default code

3
Paste the Script

Copy the entire file apps-script/Code.gs from your project → Paste it into Apps Script

4
Fill in CONFIG at the top

Edit these values in Code.gs:
• SHEET_ID — from your sheet URL: /d/THIS_PART/edit
• OWNER_EMAIL — your email for notifications
• MANAGER_PIN — same as in data/config.json

5
Deploy as Web App

Deploy → New Deployment → Web App
Execute as: Me
Who has access: Anyone
→ Click Deploy → Authorize → Copy the URL

6
Paste URL into config.json

Open data/config.json → set appsScriptUrl to the URL ending in /exec

7
Set Manager PIN

In data/config.json set managerPin to your PIN.
Also update MANAGER_PIN inside Code.gs to match.

✅ After saving config.json and pushing to GitHub, refresh this page — orders will load automatically.
